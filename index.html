Repositorios

Mis proyectos de desarrollador.

ToDoList App

Una simple lista de tareas pendientes hecho con React Hooks.
Los distintos componentes manejan el estado de los items de la lista (pueden ser agregados, eliminados y tachados para indicar que ya se completó el item) y Local Storage para que el estado de la lista se mantenga en la memoria del navegador sin importar que este se cierre o se recargue.
Tiene un estilo simple y minimalista hecho con Bootstrap.

Ver en GitHub Probar app

Gif Expert App

Mi primer app hecha desde cero con React, que es parte de los trabajos hechos en este curso de Udemy sobre React, en el que estoy aprendiendo nuevas tecnologías y adquiriendo conocimientos avanzados sobre Javascript. Es un simple buscador de GIFS hecho con la API de Giphy, Hooks y componentes hechos desde 0 por mí.
